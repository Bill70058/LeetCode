### 476.findComplement
----
#### 题目
给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。


**注意**:

- 给定的整数保证在32位带符号整数的范围内。
- 你可以假定二进制数不包含前导零位。

----
#### 示例
##### 示例 1:

```
输入: 5
输出: 2
解释: 5的二进制表示为101（没有前导零位），其补数为010。所以你需要输出2。
```

##### 示例 2:

```
输入: 1
输出: 0
解释: 1的二进制表示为1（没有前导零位），其补数为0。所以你需要输出0。
```

----
#### 解题思路
计算出目标数的二进制数长度，创建一个长度相等的数每个位都补一，最后将目标数与该数取反即可得到补数
