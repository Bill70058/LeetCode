### 852.peakIndexInMountainArray
----
#### 题目
我们把符合下列属性的数组 A 称作山脉：

A.length >= 3
存在 0 < i < A.length - 1 使得A[0] < A[1] < ... A[i-1] < A[i] > A[i+1] > ... > A[A.length - 1]
给定一个确定为山脉的数组，返回任何满足 A[0] < A[1] < ... A[i-1] < A[i] > A[i+1] > ... > A[A.length - 1] 的 i 的值。

---- 

#### 示例

##### 示例 1：

```
输入：[0,1,0]
输出：1
```

##### 示例 2：

```
输入：[0,2,1,0]
输出：1
``` 

**提示**：

- 3 <= A.length <= 10000
- 0 <= A[i] <= 10^6
- A 是如上定义的山脉

----
#### 解题思路
其实就是找最大值下标，所有搜索方法都可以
- 一、二分法
- 二、遍历找出比下一个元素大的元素返回即可(最快)
- 三、内置函数排序输出最后一个元素。
