### 172.numberFactorials
----
#### 题目
给定一个整数 n，返回 n! 结果尾数中零的数量。

----
#### 示例

##### 示例 1:

```
输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
```

##### 示例 2:

```
输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
```

**说明**: 你算法的时间复杂度应为 O(log n) 。

----
#### 解题思路
使用阶乘的定义计算阶乘会超出时间限制，使用递归计算会超过最大调用栈的大小，所以只能找规律。
找规律可以得到，2与5相乘就会末尾得到0，同理，2的倍数与5的倍数相乘也会得到0，但是2的倍数比5多，所以只需要找5的个数的就可以了，其中25可以拆分为2**5
