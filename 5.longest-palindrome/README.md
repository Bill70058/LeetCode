### 5.longest-palindrome
----
#### 题目

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。
----
#### 示例
##### 示例 1：

```
输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。
```

##### 示例 2：

```
输入: "cbbd"
输出: "bb"
```
----
#### 解题思路
中心扩展算法：回文中心的两侧互为镜像。因此，回文可以从它的中心展开，并且只有 2n - 1，2n−1 个这样的中心。
因为字符串的中心可能是两个字符，例如：``abba``，所以中心个数为2n-1
