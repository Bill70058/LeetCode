### 961.repeatedNTimes
----
#### 题目
在大小为 2N 的数组 A 中有 N+1 个不同的元素，其中有一个元素重复了 N 次。

返回重复了 N 次的那个元素。

---- 
#### 示例

##### 示例 1：

```
输入：[1,2,3,3]
输出：3
```

##### 示例 2：

```
输入：[2,1,2,5,3,2]
输出：2
```

##### 示例 3：

```
输入：[5,1,5,2,5,3,5,4]
输出：5
``` 

**提示**：

- 4 <= A.length <= 10000
- 0 <= A[i] < 10000
- A.length 为偶数

----
#### 解题思路
核心思路：重复的数必定只有两种排列方式，一种是相邻，一种是不相邻，如果相邻，那么通过遍历判断可以推出，如果不相邻，则通过遍历判断与最后一位推出。
