### 279.numSquares
----
##### 题目
给定正整数 n，找到若干个完全平方数（比如 1, 4, 9, 16, ...）使得它们的和等于 n。你需要让组成和的完全平方数的个数最少。

----
#### 示例

##### 示例 1:

```
输入: n = 12
输出: 3 
解释: 12 = 4 + 4 + 4.
```

##### 示例 2:

```
输入: n = 13
输出: 2
解释: 13 = 4 + 9.
```
----
#### 解题思路
- 一、动态规划
- 二、数学工具：四平方定理： 任何一个正整数都可以表示成不超过四个整数的平方之和。 推论：满足四数平方和定理的数n（四个整数的情况），必定满足 n=4^a(8b+7)。
