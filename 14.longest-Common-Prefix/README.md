### 14.longestCommonPrefix
----
#### 题目
编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。
----
#### 示例
----
##### 示例1

```
输入: ["flower","flow","flight"]
输出: "fl"
```

##### 示例2
```
输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。
```

----
#### 说明

默认所有输入只包含小写字母 a-z 。

#### 解题思路

将数组第一个字符串取出，将数组第一个和其他进行对比，[0,1],[0,2],[0,3]...这样一直对比下去，
其中，以第一个字符串为标准，如果找不到就将第一个字符串从后往前减一个字符，再进行对比，直到第一个字符串为空
