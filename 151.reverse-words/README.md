### 151.reverseWords
----
#### 题目
给定一个字符串，逐个翻转字符串中的每个单词。

----
#### 示例

##### 示例 1：

```
输入: "the sky is blue"
输出: "blue is sky the"
```

##### 示例 2：

```
输入: "  hello world!  "
输出: "world! hello"
解释: 输入字符串可以在前面或者后面包含多余的空格，但是反转后的字符不能包括。
```

##### 示例 3：

```
输入: "a good   example"
输出: "example good a"
解释: 如果两个单词间有多余的空格，将反转后单词间的空格减少到只含一个。
```

----
#### 解题思路
先将字符串首尾多余的空格去除，再转换为数组去除多余的空格，再拼接为字符串插入空格。
